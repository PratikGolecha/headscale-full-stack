# Headscale Full Stack Installer

Production-ready automated installer for Headscale + Headplane + Caddy

## Quick Start

```bash
curl -fsSL https://raw.githubusercontent.com/PratikGolecha/headscale-full-stack/main/install.sh | sudo bash
```

## Features

- ✅ Fully automated
- ✅ Automatic HTTPS
- ✅ Multi-distro support
- ✅ SELinux compatible
- ✅ Health checks
